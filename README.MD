# Arduino Analog Drum Machine

![alt text](https://raw.githubusercontent.com/ernesto-g/DrumMachine/master/Pictures/1.jpg)


Supported features:
  - Midi in
  - Trigger Out
  - Accent
  - Six instruments:
    - Bass Drum
    - Snare Drum
    - Hand Clap
    - Open Hi-Hat
    - Close Hi-Hat
    - Claves	
  - 32 non volatile patterns
  - 16 steps per pattern
  - Chain patterns
  - Tempo range: 30 bpm to 350 bpm
